# Landing Page

This is a landing page created with technologies like React.js and the css preprocessor, Stylus, 100% responsive.

Cuenta con pruebas de Unit Testing con Jest, las cuales se encuentran en la rama "test"

## Installation

```bash
npm install
```

## Execution

```bash
npm run start
```
